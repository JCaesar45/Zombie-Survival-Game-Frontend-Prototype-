# Zombie Survival Game (Frontend Prototype)

## Overview
This is a simple web-based zombie survival game prototype built with HTML, CSS, and JavaScript. It features user authentication, a player-controlled character, basic zombie AI, shooting mechanics, and a HUD displaying health, ammo, and score.

---

## Features
- User authentication system (login screen)
- Interactive game interface with a canvas element
- Basic player movement and shooting mechanics
- Zombies that move towards the player (simple AI)
- HUD with health bar, ammo count, and score
- Weapon selection menu
- Basic collision detection
- Score tracking

*Note:* This is a foundational prototype intended for further development and feature expansion.

---

## Setup Instructions

### Prerequisites
- Web browser (Chrome, Firefox, Edge, etc.)
- Basic knowledge of HTML, CSS, and JavaScript

### Running the game
1. Download or clone this repository.
2. Ensure the following files are in the same directory:
   - `index.html`
   - `styles.css`
   - `script.js`
3. Open `index.html` in your preferred web browser.
4. Enter a username and click "Login" to start playing.

---

## How it works
- Upon login, the game canvas appears.
- Use arrow keys to move the player.
- Click anywhere on the canvas to shoot.
- Zombies will move towards the player.
- Shooting zombies increases the score.
- Ammo decreases with each shot.
- You can select different weapons from the weapon menu (basic setup).
